# Backgammon

> Projekt pri predmetu Programiranje 2.

## Opis programa

Osnovni model je seznam števil, ki prestavljajo število kamenčkov na posameznem polju.
Pozitivna števila predstavljajo črne kamenčke, negativna števila pa bele.
